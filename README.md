# SAMA-SAMA Halal Food

Website resmi SAMA-SAMA Halal Food, Nagoya.

## Tech stack
React 19 + TypeScript + Vite + Tailwind CSS + shadcn/ui

## Jalanin di lokal
```bash
npm install
npm run dev
```

## Build production
```bash
npm run build
```
Hasilnya ada di folder `dist/`.

## Deploy
Otomatis via Vercel setiap kali push ke branch `main`.
